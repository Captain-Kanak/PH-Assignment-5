Question-1: What are the different ways to select an element in the DOM?

In JavaScript, there are multiple ways to select elements from the DOM. Here are the most common methods: getElementById, getElementsByClassName, getElementsByTagName, querySelector and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent?

1. innerHTML: Retrieves or sets the HTML content inside an element. Parses and renders HTML tags if present.

2. innerText: Retrieves or sets the visible text inside an element. Ignores hidden elements. Automatically formats text as it appears in the browser.

3. textContent: Retrieves or sets the raw text content, including hidden elements. Does not parse HTML tags; treats them as plain text.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique in JavaScript where a parent element listens for events on its child elements using event bubbling. Instead of adding event listeners to multiple child elements, you attach a single event listener to their parent.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a mechanism in the DOM where an event starts from the target element and "bubbles up" to its parent elements, one level at a time, until it reaches the root of the document.

Question-5: How do you create, add, and remove elements using JavaScript?

1. Creating Elements: To create new elements in the DOM, use the document.createElement() method.

2. Adding Elements to the DOM: Once you've created an element, you can add it to the DOM using method appendChild().

3. Removing Elements from the DOM: To remove an element, use the removeChild() or remove() methods.
